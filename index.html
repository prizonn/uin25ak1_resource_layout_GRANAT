HTML

HTML står for HyperText Markup Language, og er et strukturspråk som brukes for å lage strukturer til nettside- og applikasjonsgrensesnitt.

CSS

CSS står for Cascading StyleSheets, og brukes for å sette stilregler på HTML-elementer.

JavaScript

JavaScript er et scriptspråk basert på EcmaScript. JavaScript kjører direkte i nettleseren, og brukes ofte til å manipulere HTML og CSS i webgrensnesnitt.

React

React er et rammeverk bygget i JavaScript. React bruker komponenter og states for å lage en levende frontend.

Sanity and headless CMS

Sanity er et headless CMS som står for innholdsadministrasjon. Innhold hentes inn i applikasjoner via GROQ-spørringer.
